# Weather App

Instruções caso de erro de importação com o javafx

- Vá até Project Structure -> Libraries
- Corrija o caminho do javafx para a pasta /libs/javafx-sdk-11-0.2/lib do projeto
- Clique em OK
- Compile o projeto

Entrypoints de execução

- src/app/ui/Main
- src/app/cli/Main
